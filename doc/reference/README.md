# Gitie 参考资料

本目录包含 Gitie 项目的参考资料和补充文档，为开发者和用户提供额外的技术背景、学习资源和外部链接。这些资料有助于更深入地理解项目使用的技术、工具和最佳实践。

### 开发工具和技术
