# Gitie 项目 ROADMAP

## 简介

本文档概述了 Gitie 项目的开发路线图，重点关注 Tree-sitter 集成与 Git 工作流增强功能。此路线图将根据社区反馈和技术发展进行调整。

## 当前版本：v0.5

当前版本已完成:
- 基本的 Git 命令增强
- AI 辅助提交信息生成
- 初步的 Tree-sitter 集成
- 支持多种编程语言的语法分析（Python, Rust, Java, C/C++, Go, JavaScript）

## 未来版本规划

### v0.6 - 代码评审与命令集成 (短期目标)

**主题**: 构建代码评审基础框架并集成到提交流程

#### 功能

1. **代码评审基础功能**
   - 实现 `gitie review [options] [<commit>] [<commit>]` 命令
   - 基于 Tree-sitter 的结构化代码分析
   - 多维度评审（代码结构、语法、风格）
   - 添加评审结果的多种输出格式（文本、JSON、HTML）

2. **结构化差异分析**
   - 按函数/类/方法展示代码变更
   - 识别移动的代码块而非仅显示删除和添加
   - 提供更有意义的变更摘要

3. **评审与提交集成**
   - 新增 `gitie commit --review` 选项
   - 提交前自动进行代码评审
   - 交互式评审结果处理
   - 评审问题修复建议

4. **统一参数系统**
   - 完善的长参数支持（如 `--review`, `--ts` 用于 tree-sitter）
   - 参数组合支持（如 `--review-ts`）
   - 与原生 Git 命令兼容的参数设计

5. **语言专用安全审核提示集成**
   - 利用已配置的语言专用安全审核提示（存放于 assets/review-<language>-prompt.md）
   - 针对不同编程语言（Python, Rust, Java, C/C++, Go, JavaScript）的专用安全框架
   - 根据文件类型自动加载相应语言的安全审核提示
   - 语言特定的安全最佳实践和常见漏洞模式

6. **配置系统增强**
   - 评审深度配置（基础、标准、深入）
   - 评审关注点配置（安全、性能、风格、语义）
   - 支持项目级和全局配置文件

**实现复杂度**: 中等
**预计交付时间**: 4-5 周

---

### v0.7 - 多语言支持与性能优化 (中期目标)

**主题**: 扩展语言支持，优化性能，增强安全分析

#### 功能

1. **多语言分析增强**
   - 优化所有支持语言的分析引擎
   - 语言特定规则库
   - 跨语言共通问题检测

2. **性能优化**
   - 增量分析支持
   - 并行处理大型代码库
   - 分析结果缓存

3. **代码质量分析**
   - 计算和报告复杂度指标（圈复杂度、认知复杂度等）
   - 跟踪代码质量随时间的变化趋势
   - 识别需要重构的代码热点

4. **高级报告功能**
   - HTML 报告生成
   - 可视化代码变更
   - 问题导航和筛选
   - 报告导出（JSON、PDF）

5. **基础安全漏洞检测**
   - 检测常见的安全问题模式
   - 识别敏感信息暴露
   - 对权限和访问控制变更提供警告

**实现复杂度**: 高
**预计交付时间**: 4-6 周

---

### v0.8 - 增强 AI 集成与高级评审 (长期目标)

**主题**: 加强 AI 能力，提高代码理解和评审质量

#### 功能

1. **深度 AI 代码语义理解**
   - 上下文感知的代码分析
   - 跨文件依赖分析
   - 自适应代码意图推断

2. **智能重构建议**
   - 基于 AST 分析生成特定重构建议
   - 提供自动应用简单重构的选项
   - 设计模式识别与优化建议

3. **预提交评审工作流优化**
   - 交互式问题修复
   - 智能建议排序
   - 学习用户偏好

4. **API 变更检测**
   - 识别公共 API 的变更
   - 突出显示破坏兼容性的修改
   - 推荐版本更新策略（根据语义化版本规范）

5. **团队协作增强**
   - 集成到 PR/MR 工作流
   - 生成团队编码规范遵守报告
   - 支持项目特定的分析规则

**实现复杂度**: 非常高
**预计交付时间**: 8-10 周

---

### v0.9 - 企业与团队功能 (愿景目标)

**主题**: 添加企业级功能和团队协作支持

#### 潜在功能

1. **团队代码规范检查**
   - 可定制的规范检查规则
   - 团队特定规则库
   - 规范遵守报告和趋势

2. **高级安全分析**
   - 深入的静态代码安全分析
   - 依赖安全审计
   - 符合行业标准的安全报告

3. **系统架构分析**
   - 基于代码变更推断系统架构
   - 生成架构图和依赖关系
   - 架构演进追踪

4. **与 CI/CD 系统深度集成**
   - 作为 CI 流水线的一部分运行
   - 提供质量门控功能
   - 生成历史趋势报告

5. **性能问题识别**
   - 检测潜在的性能瓶颈
   - 提供优化建议
   - 支持基准比较

**实现复杂度**: 极高
**预计交付时间**: 10-12 周

### v1.0 - 稳定版本 (最终目标)

**主题**: 整合所有功能，优化用户体验，确保稳定性

#### 潜在功能

1. **全功能集成与稳定**
   - 全面功能测试与稳定
   - 性能优化
   - 用户体验完善

2. **自定义规则引擎**
   - 允许用户创建特定领域规则
   - 规则编辑器和测试工具
   - 规则共享和社区库

3. **完整的文档和示例**
   - 全面的用户手册
   - 丰富的示例集合
   - 视频教程

4. **社区贡献框架**
   - 插件系统
   - 扩展 API
   - 贡献者指南

5. **国际化与本地化**
   - 多语言支持
   - 适应不同区域的编码规范

**实现复杂度**: 高
**预计交付时间**: 6-8 周
